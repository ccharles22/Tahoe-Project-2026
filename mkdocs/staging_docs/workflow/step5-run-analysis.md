# Step 5: Run Analysis

Generate visualizations and analysis reports for your experiment.

## Overview

This step:

- Creates activity distribution plots
- Generates lineage trees
- Computes top 10 variants
- Builds protein similarity networks
- Optionally creates bonus visualizations

## Prerequisites

- ✅ **Completed Step 4** (Process Sequences)

## Generated Outputs

### Standard Visualizations

1. **Activity Distribution** - Histogram of activity scores across all variants
2. **Lineage Tree** - Directed graph showing variant ancestry
3. **Top 10 Variants** - Bar chart and CSV of best performers
4. **Protein Similarity Network** - Network graph based on sequence similarity

### Bonus Visualizations (Optional)

5. **Activity Landscape** - 3D PCA surface plot
6. **Mutation Fingerprints** - Interactive heatmaps per variant
7. **Domain Enrichment** - Heatmap of mutations by protein domain
8. **Mutation Frequency** - Position-wise mutation frequency plots

## Running Analysis

1. Locate **Step 5** in the sidebar
2. Click **Run Analysis**
3. Wait for processing to complete
4. View generated plots in the workspace panels

## Downloading Results

All outputs are available for download from the workspace:

- PNG images for presentations
- CSV files for further analysis
- Interactive HTML visualizations

## Next Steps

Analysis complete! Review your results and explore the visualizations.
