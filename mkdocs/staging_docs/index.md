# Direct Evolution Monitoring Portal - User Guide

Welcome to the Direct Evolution Monitoring Portal user guide! This guide will walk you through the complete workflow for processing and analyzing directed evolution experiments.

## What is the Direct Evolution Monitoring Portal?

The Direct Evolution Monitoring Portal is a comprehensive web-based platform designed to help researchers analyze directed evolution experiments. It provides tools for:

- **Wild-type protein retrieval** from UniProt
- **Plasmid sequence validation** against wild-type proteins
- **Variant data upload** and parsing
- **Sequence processing** and mutation calling
- **Automated analysis** and visualization generation

## Quick Start

The portal follows a 5-step workflow:

1. **Fetch Wild-Type (UniProt)** - Retrieve the reference protein sequence from UniProt
2. **Validate Plasmid** - Upload and validate your plasmid FASTA file
3. **Upload Data** - Upload your variant data (TSV, CSV, or JSON)
4. **Process Sequences** - Automatically translate DNA sequences and call mutations
5. **Run Analysis** - Generate visualizations and analysis reports

Each step builds on the previous one, creating a complete analysis pipeline for your directed evolution experiment.

## Getting Started

To begin using the portal:

1. Navigate to the **Workspace** page from the main navigation
2. Start with **Step 1: Fetch Wild-Type** in the left sidebar
3. Follow the workflow steps in order

!!! tip "Guest Access"
    You can use the portal as a guest without signing in. Your work will be preserved during your session, but we recommend creating an account to maintain long-term access to your experiments.
