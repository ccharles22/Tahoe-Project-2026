## MkDocs Layout

- `parsing_qc/`: Parsing/QC docs, config, and built site
- `postgresql_visualization/`: PostgreSQL visualization docs, config, and built site

### Preview locally

```bash
cd mkdocs/parsing_qc
mkdocs serve -a 127.0.0.1:8000
```

```bash
cd mkdocs/postgresql_visualization
mkdocs serve -a 127.0.0.1:8001
```
