# Bonus Visualisations

The bonus visualisation guide covers the optional Section 6 outputs generated
after the main analysis completes. These views extend the core result
visualisations with deeper exploratory plots for mutation patterns and advanced
comparisons.

## Included views

- **Activity Landscape**: interactive embedded landscape view for mutation
  embeddings and activity patterns.
- **Mutation Fingerprinting**: per-variant mutation profile across the protein
  sequence.
- **Domain Enrichment Heatmap**: domain-level mutation enrichment summaries.
- **Mutation Frequency**: position-based mutation hotspot overview.

## When to use this section

Use the bonus guide when you want to understand the advanced Section 6 outputs
shown in the experiment workspace, including what each plot means, what data it
depends on, and how it should be interpreted.
