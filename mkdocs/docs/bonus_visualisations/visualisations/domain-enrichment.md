# Domain Enrichment Heatmap

The Domain Enrichment Heatmap summarizes how mutation burden is distributed
across annotated protein regions.

## What it shows

- which domains accumulate more non-synonymous changes
- how mutation density differs between protein regions
- how domain-level mutation load changes across generations

## Why it is useful

This view helps distinguish broad sequence drift from focused regional change,
which is useful when interpreting whether evolution is concentrating on specific
functional parts of the protein.
