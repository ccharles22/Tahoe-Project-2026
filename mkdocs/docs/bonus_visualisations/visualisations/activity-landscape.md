# Activity Landscape

The Activity Landscape is an interactive view that maps variants in a reduced
embedding space and compares those positions against activity-related values.

## What it shows

- relative clustering of variants
- how high- and low-performing variants distribute across the embedding
- broad performance regions rather than only rank-order summaries

## Why it is useful

This view helps users see whether strong variants form clear clusters or whether
performance is scattered across many mutation patterns.

## When it may be unavailable

This plot depends on valid activity-related metrics. If those are not available
for the latest generation, the workspace shows a placeholder explaining why the
plot was skipped.
