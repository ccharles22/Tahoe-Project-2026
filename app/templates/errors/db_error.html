Database Unavailable

The PostgreSQL server couldn't be reached. This usually means the database host is offline or the network connection timed out.

Things to check:
• Is the database server running?
• Are you connected to Tailscale / VPN?
• Can you reach 100.80.183.102:5432?
• Try: pg_isready -h 100.80.183.102 -p 5432
