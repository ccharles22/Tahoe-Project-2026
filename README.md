# MariaDB backend

This repository contains the MariaDB backend for the Directed Evolution Monitoring Web Portal, containing all the data precomputed and computed.

Key points:
- Database schema design (SQL in MariaDB)
- Unified activity metric computation
- Precomputed analysis tables (MariaDB)
- Variant lineage modelling (OQGRAPH)
- Data segregation (step 5 following MariaDB)
