📋 Expected Data Format

Your data file should contain the following fields:

  • variant_index - Unique identifier for the variant (integer)
  • generation - Generation number (integer, starting from 0)
  • parent_variant_index - Parent variant ID (integer, or empty for generation 0)
  • assembled_dna_sequence - DNA sequence string (A, T, G, C characters)
  • dna_yield - DNA yield measurement (numeric)
  • protein_yield - Protein yield measurement (numeric)

Additional columns will be stored as metadata.

ℹ️ How It Works
  1. Upload your data file (TSV, CSV, or JSON format)
  2. The system parses and validates each record
  3. Quality control checks are performed:
    • Required fields presence
    • Data type validation
    • Value range checks
    • DNA sequence validation
  4. Valid records are stored in the database
  5. You'll see a summary with any warnings or errors
